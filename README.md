


djElenor App
==================


This is the repository for the [djElenor]
app. It is a **work in progress** 🚧.

**djElenor** is a fully functional Django app built  with Django and Graphql. It
follows Django design and development best practices.

# UI
The app was designed using [Material 3 guidelines](https://m3.material.io/). 

The Screens and UI elements are built  using [Django Template](https://docs.djangoproject.com/en/5.0/). 




## Android App Storefront

An open-source storefront in React.js with Next.js. Built for Headless Commerce, using a modern stack with TypeScript, GraphQL, Apollo, and Tailwind CSS.
